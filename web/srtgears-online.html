Srtgears online

This page is an online Srtgears interface. You can also download the offline Srtgears command line tool.

Input form

You only have to fill the fields you want to perform on your subtitle files. Also be aware of Srtgears' limits.

  • output file name (*.srt or *.ssa)
  • optional 2nd output file name (when splitting) (*.srt or *.ssa)
  • concatenate 2 subtitle files, 2nd part start at e.g. ''00:59:00,123'
  • merge 2 subtitle files (first at bottom, second at top)
  • lengthen / shorten display duration of subtitles, multiplier e.g. for +10% use 1.1
  • remove controls such as {\anX} (or {\aY}, {\pos(x,y)})
  • remove hearing impaired subtitles (such as '[PHONE RINGING]' or '(phone ringing)')
  • strip off formatting (such as <i>, <b>, <u>, <font>)
  • change subtitle position
  • change subtitle color, name (e.g. 'red' or 'yellow') or RGB hexa '#rrggbb' (e.g. '#ff0000' for red)
  • scale subtitle timestamps (faster/slower); multiplier e.g. 1.001
  • shift subtitle timestamps (+/- ms)
  • time where to split to 2 subtitle files (first and second output files), e.g. '00:59:00,123'
  • analyze file and print statistics
